# 📓 Agenda de Contatos | Desafio Rocketseat

Este projeto é uma agenda de contatos desenvolvida em Python, baseada no desafio do módulo "Introdução ao Python" da Rocketseat. A aplicação é executada inteiramente no terminal.

## 🚀 Funcionalidades

O objetivo do desafio era desenvolver uma agenda com as seguintes funcionalidades:

* [x] **Mostrar um menu de opções:** Permitir que o usuário escolha a ação que deseja realizar.
* [x] **Adicionar contato:** Implementar a funcionalidade para adicionar um novo contato (Nome, Telefone, Email).
* [x] **Listar contatos:** Desenvolver a visualização da lista de contatos cadastrados.
* [x] **Editar contato:** Criar a funcionalidade para editar um contato existente.
* [x] **Marcar/Desmarcar favorito:** Implementar a opção para marcar ou desmarcar um contato como favorito.
* [x] **Listar favoritos:** Desenvolver a visualização da lista de contatos favoritos.
* [x] **Apagar contato:** Criar a funcionalidade para apagar um contato.

## 🛠️ Tecnologias Utilizadas

* **Python**
